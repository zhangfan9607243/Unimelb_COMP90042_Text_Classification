## Project Introduction

This project aims to implement a text classification system to predict the country of origin of a given tweet, through a series of tasks including preprocessing, feature extraction, classifier building and evaluation.

## Project Tasks

The project is structured into the following tasks:

- Part 1: Preprocessing

  - Task 1.1: Bag-of-Words Representation
  - Task 1.2: Hashtags Tokenization - MaxMatch
  - Task 1.3: Hashtags Tokenization - Reversed MaxMatch
  - Task 1.4: Unigram Language Model for Hashtag Scoring

- Part 2: Text Classification

  - Task 2.1: Data Partitioning and Feature Extraction
  - Task 2.2: Building Classifiers and Hyper-parameter Tuning
  - Task 2.3: Test Performance Comparison
  - Task 2.4: Feature Analysis
